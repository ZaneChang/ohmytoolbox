# OhMyToolbox Supabase 后端设置

## 步骤1: 注册 Supabase

1. 打开 https://supabase.com
2. 点击 "Start your project"
3. 用 GitHub 账号登录
4. 创建新项目: "ohmytoolbox"

## 步骤2: 创建数据表

在 Supabase 后台:

### 2.1 创建 ai_tools 表
```sql
create table ai_tools (
  id bigint generated by default as identity primary key,
  name text not null,
  name_zh text,
  desc text,
  name_zh_desc text,
  type text default 'ai',
  icon text default '📌',
  hot boolean default false,
  link text not null,
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```

### 2.2 创建 tg_channels 表
```sql
create table tg_channels (
  id bigint generated by default as identity primary key,
  name text not null,
  name_zh text,
  desc text,
  name_zh_desc text,
  type text default 'tg',
  icon text default '📌',
  hot boolean default false,
  link text not null,
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```

### 2.3 创建 productivity 表
```sql
create table productivity (
  id bigint generated by default as identity primary key,
  name text not null,
  name_zh text,
  desc text,
  name_zh_desc text,
  type text default 'productivity',
  icon text default '📌',
  hot boolean default false,
  link text not null,
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```

## 步骤3: 获取 API URL

1. 在 Supabase 后台 → Settings → API
2. 复制 "Project URL"
3. 复制 "anon public" key

## 步骤4: 填入数据

在每个表中添加数据(图形界面操作):
- name, name_zh, desc, name_zh_desc, type, icon, hot, link

## 步骤5: 配置权限

Tables → 选择表 → Edit table → 开启:
- Enable Row Level Security (关闭)
- Allow public read (开启)

或者运行:
```sql
alter table ai_tools enable row level security;
create policy "Public read" on ai_tools for select using (true);
create policy "Public insert" on ai_tools for insert with check (true);
create policy "Public update" on ai_tools for update using (true);
create policy "Public delete" on ai_tools for delete using (true);
-- 同样的为 tg_channels 和 productivity
```
