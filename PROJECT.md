# OhMyToolbox - 项目规划

## 域名
www.ohmytoolbox.com

## 定位
一个聚合型工具导航网站，包含：

### 第一期功能
1. **AI工具导航** - 收录主流AI工具
2. **TG群组/频道收录** - Telegram群组导航
3. **OpenClaw Skills推荐** - 精选AI助手技能

### 后期可扩展
- Prompt库
- AI资讯/博客
- 付费社群

## 技术栈
- Next.js 15 + TypeScript
- Tailwind CSS
- App Router

## 页面结构
```
/                     首页 - 工具箱入口
/ai-tools            AI工具导航
/tg-groups            TG群组收录
/skills               OpenClaw Skills推荐
/about                关于页面
```

## 设计风格
- 简洁现代
- 卡片式布局
- 暗色主题为主
- 强调交互体验
